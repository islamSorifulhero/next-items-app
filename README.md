# Next Items App

## Description
A simple Next.js 15/16 application with mock authentication, public items listing, item details, and a protected add-item page.

## Features
- Landing Page with 7 sections
- Mock Login (email/password)
- Items List & Details Page
- Protected Add Item Page
- Toast notifications for success
- Responsive UI with Tailwind CSS & DaisyUI

## Setup & Installation

### Backend
```bash
cd backend
npm install
nodemon index.js
